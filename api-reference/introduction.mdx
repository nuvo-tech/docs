# Nuvo Public API

The Nuvo Public API allows you to programmatically interact with Nuvo services.

## Authentication

All API requests require authentication using an API key. Include your API key in the `Authorization` header using the Bearer scheme:

```
Authorization: Bearer sk_prod_your_api_key_here
```

API keys are environment-specific and prefixed accordingly:
- **Staging**: `sk_stg_`
- **Production**: `sk_prod_`

## API Versioning

Specify the API version using the `Nuvo-API-Version` header. This header is required for all API requests.

```
Nuvo-API-Version: 01-05-2026
```

Currently supported versions: `01-05-2026`

For API support, please contact your Nuvo representative or Customer Success team.
